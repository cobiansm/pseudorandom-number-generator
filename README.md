# AES-128-CBC Encryption on FRDM-K64

## Overview

This project implements the **AES-128 (Advanced Encryption Standard)** block cipher in **CBC (Cipher Block Chaining)** mode on the **FRDM-K64** development board using the **mbedTLS** library and the **MCUXpresso SDK**. The application demonstrates the complete cryptographic workflow, including the initialization of a 128-bit key and Initialization Vector (IV), the encryption of a 32-byte plaintext, and the subsequent decryption to recover the original message.
